# Dashboard Feature (대시보드)

## 📌 개요

사용자의 전체적인 부수입 현황을 한눈에 파악할 수 있도록 데이터를 집계하여 제공하는 기능입니다.
별도의 데이터베이스 테이블을 가지지 않고, `Transaction` 데이터를 가공하여 결과를 반환합니다.

## 🏗 구조

- **Controller**: `DashboardController`
  - 대시보드 UI에 필요한 다양한 통계 데이터를 위한 전용 API 엔드포인트 제공.
- **Service**: 별도의 Service 없음 (`TransactionService` 활용)
  - `TransactionService`에 구현된 집계 메서드들을 호출하여 데이터를 조합합니다.

## 🔑 주요 역할

1. **종합 요약**: 이번 달 총 수익, 순수익, 지출, 투입 시간 및 전월 대비 증감률 제공.
2. **포트폴리오 분석**: 어떤 수입원이 전체 수익에서 얼마나 비중을 차지하는지 분석.
3. **랭킹 시스템**: 가장 수익 효율이 좋은 수입원 순위 제공.
4. **월별 추이**: 전체적인 수입 성장을 그래프로 시각화하기 위한 데이터 제공.
