# Income Source Feature (수입원 관리)

## 📌 개요

사용자의 다양한 부수입원(예: 사이드 프로젝트, 외주, 투자 등)을 관리하는 핵심 기능입니다.
각 수입원은 `Transaction`(거래 내역)과 1:N 관계를 가집니다.

## 🏗 구조

- **Controller**: `IncomeSourceController`
  - 수입원 CRUD 및 관련 통계/거래 내역 조회 API 제공.
- **Service**: `IncomeSourceService`
  - 수입원 생성, 수정, 삭제, 조회 비즈니스 로직 처리.
- **Entity**: `IncomeSource`
  - 수입원 데이터 모델 정의 (이름, 설명, 색상 등).

## 🔑 주요 역할

1. **수입원 관리**: 새로운 수입원을 등록하거나 정보를 수정/삭제합니다.
2. **거래 내역 연동**: 특정 수입원에 연결된 모든 거래 내역을 조회합니다.
3. **통계 제공**: 수입원별 월별 수익/지출 현황 및 요약 정보를 제공합니다.
