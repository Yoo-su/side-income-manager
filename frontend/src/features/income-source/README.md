# Income Source Feature (Frontend)

## 📌 개요

수입원 목록을 조회하고, 새로운 수입원을 추가하거나 관리하는 UI를 제공합니다.

## 🏗 주요 컴포넌트

- **IncomeSourceList**: 등록된 수입원들을 목록(List) 또는 그리드(Grid) 형태로 표시합니다.
- **IncomeSourceCard**: 개별 수입원 카드로, 이름, 설명, 태그 등을 포함합니다.
- **IncomeSourceFormDialog**: 수입원 생성 및 수정을 위한 모달 폼(Dialog)입니다.
- **IncomeSourceDetail**: 특정 수입원의 상세 정보와 해당 수입원의 거래 내역을 보여주는 페이지입니다.

## 🔑 주요 역할

1. **CRUD 인터페이스**: 사용자가 직관적으로 수입원을 관리할 수 있는 UI를 제공합니다.
2. **상세 정보 제공**: 각 수입원별로 누적 수익, 평균 수익 등 상세한 성과 지표를 확인할 수 있습니다.
3. **필터링 연동**: 거래 내역 조회 시 특정 수입원을 기준으로 필터링하는 기능을 지원합니다.
4. **미니 차트 필터링**: 상세 페이지의 월별 추이 차트에서 기간을 선택하여 확인할 수 있습니다.

## 📱 반응형 동작

- **목록**: Desktop(3열) → Tablet(2열) → Mobile(1열) 카드 그리드로 전환됩니다.
- **상세 페이지**: 요약 카드가 화면 너비에 따라 최적화된 그리드(5열 → 3열 → 2열)로 배치됩니다.
