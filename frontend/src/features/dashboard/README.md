# Dashboard Feature (Frontend)

## 📌 개요

애플리케이션의 홈 화면으로, 사용자의 수익 현황을 직관적인 UI로 제공합니다.

## 🏗 주요 컴포넌트

- **StatsCards**: 핵심 지표(총 수익, 순수익 등)를 카드 형태로 요약하여 보여줍니다.
- **MonthlyChart**: 월별 수익/지출 추이를 막대그래프로 시각화합니다.
- **RecentTransactions**: 최근 발생한 거래 내역 리스트를 간략하게 보여줍니다.
- **PortfolioChart**: 수입원별 수익 비중을 원형 차트(Pie Chart) 등으로 시각화합니다.

## 🔑 주요 역할

1. **데이터 시각화**: 복잡한 거래 데이터를 그래프와 차트로 변환하여 정보 전달력을 높입니다.
2. **현황 파악**: 사용자가 현재 재정 상태를 빠르게 인지하고 의사결정을 내릴 수 있도록 돕습니다.
3. **빠른 접근**: 최근 내역이나 주요 수입원으로 바로 이동할 수 있는 진입점 역할을 합니다.
